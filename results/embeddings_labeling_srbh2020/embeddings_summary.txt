--- Nota metodologica ---
Esta tecnica compara embeddings de cada peticion HTTP con prototipos textuales asociados a cada categoria CAPEC. El modelo se usa congelado y no se entrena un clasificador final. Si se proporciona train_set, este se usa solo para calibrar umbrales por etiqueta, manteniendo eval_set como conjunto comun de evaluacion para la comparativa. La calibracion incorpora un limite de predicciones por etiqueta para reducir sobreetiquetado en categorias semanticamente amplias.
