# Data

The original SR-BH 2020 multi-label dataset is not included in this repository due to its file size.

The dataset can be accessed through its persistent DOI:

https://doi.org/10.7910/DVN/OGOIXX

Direct Harvard Dataverse page:

https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/OGOIXX

The scripts in this repository expect the original CSV file to be available locally as:

`data_capec_multilabel.csv`

Generated subsets and derived outputs are included in the repository.
